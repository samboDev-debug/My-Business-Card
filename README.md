# Business Card App 📇

A simple and elegant Android app built with **Kotlin** that allows users to create and share digital business cards.  
This project demonstrates modern Android development practices including **Jetpack Compose**, **Material Design**, and **MVVM architecture**.

---

## ✨ Features
- Create a personalized business card with:
  - Name
  - Job title
  - Contact information
  - Social media links
- Clean and responsive UI using **Jetpack Compose**
- Dark mode support
- Share your card easily with others

---

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM
- **Design:** Material Design 3
- **Tools:** Android Studio, Gradle
